Perguntas frequentes

Preciso criar uma conta para consultar?

Não. A prévia pode ser feita sem cadastro. Criar uma conta é útil para acompanhar seu histórico de consultas e pagamentos.

O que aparece na prévia gratuita?

Mostramos parte do nome, situação cadastral e localização aproximada. Dados sensíveis como CPF completo, telefone, e-mail e endereço ficam bloqueados até a liberação.

Como funciona o pagamento?

O pagamento é feito via Pix, de forma segura, e a liberação do resultado completo acontece automaticamente após a confirmação.

Os dados são atualizados?

Mantemos um cache local para acelerar novas consultas ao mesmo CPF, com renovação periódica junto à fonte de dados configurada.

Meus dados estão seguros?

Sim. Tratamos as informações conforme a LGPD e não compartilhamos dados de consulta com terceiros fora da finalidade do serviço.
