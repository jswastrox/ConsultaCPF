Política de Privacidade

Esta política descreve como o {{ site_name }} trata dados pessoais, em conformidade com a Lei Geral de Proteção de Dados (LGPD).

1. Dados que coletamos

Coletamos o CPF informado na busca, dados de cadastro (nome, e-mail, telefone) quando você cria uma conta, e dados técnicos de navegação (IP, user-agent) para segurança e prevenção de abuso.

2. Como usamos os dados

Usamos os dados para processar sua consulta, gerenciar sua conta, processar pagamentos via Pix e cumprir obrigações legais.

3. Compartilhamento

Não vendemos dados pessoais. Compartilhamos informações apenas com processadores de pagamento (Pix/Woovi) na medida necessária para concluir a transação.

4. Seus direitos

Você pode solicitar acesso, correção ou exclusão dos seus dados pessoais a qualquer momento, entrando em contato pelo e-mail {{ contact_email }}.

5. Segurança

Adotamos medidas técnicas razoáveis para proteger os dados armazenados, incluindo senhas com hash e sessões autenticadas por token.
